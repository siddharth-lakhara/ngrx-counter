# NG Rx Counter

A simple counter app build to learn NG Rx. 

## Features
1. Increment Counter
2. Decrement Counter
3. Reset Counter
4. Set counter value

## Screen shot
![App screenshot](./app_screenshot.png?raw=true)

## Getting started
1. Clone Repo
2. Run `npm install`
3. Run `npm start`
4. Navigate to http://localhost:4200 in your favourite browser
